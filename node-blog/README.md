#简单markdown博客
---

**涉及框架与组件**

+ express4
+ marked
+ pygmentize-bundled

**目录文件**

+ blogs
	+ yyyy-mm-dd-blog-title.md
+ theme
	+ default
+ views
	+ article.jade
	+ head.jade
	+ index.jade
+ app.js

**启动项目**

	nodemon app.js
	
**访问路径**

	http://localhost:3000

-

	http://localhost:3000/2014/11/03/blog-title
